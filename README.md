Kodlama %20 olarak tamamlanmıştır.

Özellikler -
Zamana bağlı olarak,
* Güç kapatma
* Yeniden başlatma
* Mesaj gösterme
* İnternet erişimini sınırlandırma
* İnternet erişimini sağlama

Diğer versiyon için,
